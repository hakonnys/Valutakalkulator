# Valutakalkulator
denne ble ikke sendt inn fra den stasjonere så jeg måtte sende den i norsk pausen

En enkel og brukervennlig valutakalkulator som lar deg konvertere beløp mellom ulike valutaer ved hjelp av ExchangeRate-API.
Dette prosjektet er laget med HTML, CSS og JavaScript. Du kan velge hvilken valuta du vil konvertere fra og til, skrive inn beløp, hente sanntids valutakurser fra API-et og se resultatet direkte på siden.
Valutakalkulatoren bruker ExchangeRate-API – https://www.exchangerate-api.com/

Her kan du hente valutakurs direkte fra API når du klikker på Konverter-knappen
Lagre opptil 20 favorittvalutaer som alltid vises øverst i menyene


Bruker veiledning -->
1. Skriv inn beløpet du vil konvertere i feltet Skriv beløp
2. Velg valuta fra og til fra nedtrekksmenyene
3. Klikk på Konverter-knappen
4. Resultatet vises under knappen med beløp og kursinformasjon
5. Klikk på ❤️ knappen ved siden av en valuta for å legge den til i favoritter
